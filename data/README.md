## Attendance Dataset
This folder contains a synthetic attendance dataset used for proxy attendance detection.
### Columns
- Roll_number: Student roll number
- Date: Attendance date
- Time: Attendance time
- Subject: Subject name
- Status: Present / Absent
- Label:
  - 0 = Normal attendance
  - 1 = Suspicious / Proxy attendance

### Notes
- Proxy cases are based on repeated timings and unusual patterns.
